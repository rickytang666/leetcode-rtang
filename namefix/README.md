# fix folder names

pads leetcode problem folders to 4 digits.

## usage

### mac/linux

```bash
./namefix/namefix.sh
```

### windows

```powershell
namefix\namefix.ps1
```

## examples

`56-problem-haha` → `0056-problem-haha`

output:

```
Wrong: 56-problem-haha
Fixed: 0056-problem-haha

✓ Fixed 1 folder(s)
```
